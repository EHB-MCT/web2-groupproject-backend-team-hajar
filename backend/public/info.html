Web 2 - S7: Full stack Teamwork - Challenges API

  • GET / - Shows documentation
  • GET /challenges - Get all challenges
  • GET /challenges/id - Get a challenge with the specified id in the url string - ex: challenges/61a64cb2497fe178b6c440b0
  • POST /saveChallenges - Saves a challenge to the database
  • PUT /updateChallenges - Updates a challenge in the database
  • DElETE /deleteChallenges - Deletes a challenge from the database
